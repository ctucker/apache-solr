                    Apache Solr - solr-ruby and flare Version 1.3-dev
                            Release Notes

Introduction
------------
solr-ruby is a Ruby client for the solr search engine.
flare is a plugin adding faceted browsing, AJAX suggest and more to Rails controllers.
